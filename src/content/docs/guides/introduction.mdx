### What Is Vue
    Vue (pronounced /vjuÀê/, like **view**) is a JavaScript framework for building user interfaces. It uses HTML, CSS, and JavaScript to provide a declarative, component-based framework for building complex UIs.
    
    Here is a minimal example:
```js
    import { createApp } from 'vue'

    createApp({
  data() {
    return {
      count: 0
    }
  }
    }).mount('#app')
```

```js
    <div id="app">
  <button @click="count++">
    Count is: {{ count }}
  </button>
 </div>
 ```
